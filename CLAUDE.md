# Ламопад — Artist Website

## What we're building
A personal/promo website for **Ламопад** — a Russian indie musician (modern rock + electronic experiments). Pure HTML5 + JS + CSS, bundled with Webpack 5.

## Artist identity
- **Archetype:** Wanderer/Seeker. Secondary: Sage + Trickster.
- **Narrative:** always on the road, light pack, no return ticket. Joyful absurdism meets deep longing.
- **Themes:** escape from routine · travel adventures · homesickness · search for meaning · existential absurdism
- **Language:** Russian-speaking audience

## Target audience
- **18–25** — finding themselves
- **25–35** — crisis of routine
- Core values: freedom, meaning, depth. Open to new things. Existentially inclined.

## Visual / aesthetic
- Surrealist, absurdist oil-painting cover art (Magritte style)
- Artist: red Converse, poncho, dreadlocks, red backpack
- Lives in a motorhome. Always in transit.
- Recurring symbols: llamas, the road, the red backpack

## Musical references
- Yuri Vizbor — but modern, with arrangements and post-irony
- Trofimov (trains, planes) — but smarter
- Soviet bardic / campfire songs — contemporary and fun
- Korzh — but no street vibe, more rock-reggae
- Anacondaz (trains)

## Film references (tone/mood)
- Трасса 60 (Route 60)
- Достучаться до небес (Knockin' on Heaven's Door)
- Капитан фантастик (Captain Fantastic)

## Anti-audience (who we're NOT targeting)
- Ольга, 45 — Retro FM, Odnoklassniki, loves stability
- Успешный Дмитрий, 30 — hustle-culture podcasts, considers Dubai "travelling"
- Петрович — Butyurka on the UAZ, never leaves his taiga
- Репер Derzkiy — genre purist

## Tech stack
- Pure HTML5 + vanilla JS + CSS (no framework)
- Webpack 5: `npm start` (dev server) / `npm run build` (prod → `dist/`)
- Entry: `js/app.js` · Styles: `css/style.css`
